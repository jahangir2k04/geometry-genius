Questions and Answer

1. What is DOM and what is the purpose of it?

Answer : The Document Object Model (DOM) is a programming interface for web document. We can change the document structure, style and content. It represents the document as nodes and objects.

2. How will you select HTML elements using DOM? Name the DOM methods.

Answer : There are 5 ways to select HTML elements using DOM.

  1. getsElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. quetySelector
  5. quetySelectorAll

3. What is event bubble?

Answer : Event bubbling is a concept of DOM . It is the event that starts from the target element to the top element.

3. What is a callback function and why will you use it?

Answer : A callback function is a function passed as an argument to the another function . It allows a function to call the another function. A callback function is run when another function finished his task.
